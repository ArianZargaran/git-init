Staging area

Local repository

git reset ––soft HEAD~

git add options

Description

Add changes to the staging area

Options

.
Add all the files with changes
folder
Add all the changes inside a folder. Including the folder itself.
file
Add all the changes in a file.
–u
Add all the changes of the tracked files.

git commit -m message

Description

Wrap all current changes in staging area into a commit node with the message passed as option.
